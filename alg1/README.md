# Algebra 1 Problem Bank Catalog

This catalog was rebuilt from the verified Algebra 1 item-bank archive using the cleaner CC1-style architecture.

## Structure

- `index.html` — catalog app
- `css/styles.css` — app styles
- `js/app.js` — filtering, preview, edit, new question, worksheet, and export logic
- `data/catalog_index.js` — lightweight searchable index
- `data/*.js` — one smaller data file per unit/section group
- `assets/` — original problem images plus reusable worksheet workspace images
